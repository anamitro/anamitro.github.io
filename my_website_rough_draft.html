IBus-Sasankadeva
IBus-Sasankadeva is an opensource input engine for Bengali and other Indic scripts based on the IBus Table input method in Linux. The method is specifically alphasyllabic (lipi), in the sense that the default consonant contains an implicit 'a' vowel. The main repository is regularly maintained by me.

For more details, see the official website https://sites.google.com/view/sasankadeva
